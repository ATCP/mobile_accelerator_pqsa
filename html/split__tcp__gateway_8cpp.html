enum PHASE

phases the mobile accelerator sender can be in.

similar to TCP sending phase, such as slow start, loss recovery, etc similar to TCP congestion avoidance phase but the sending rate is adjusted by the bandwidth estimation and adaptation transmitting the missing packet obsolete parameter enter this phase when timeout and use F-RTO to detect unnecessary timeout and retransmit
